# WasteRecords-dotnetCoreAPI
Waste Records Dotnet 6.0 Core API and MVC Project (job interview case study)

## Project Technology stack is
- Server code: `.Net 6.0`
- Database: `MSSQL`
- ORM: `EntityFramework Core`
- Web Framework `.Net MVC`

- `JWT`
- `AutoMapper`
- `FluentValidation`
- `Swagger`
- `RestSharp`
- `Bootsrap`, `SweetAlert`
